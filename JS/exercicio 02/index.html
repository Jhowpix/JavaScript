A const nao podem ser redeclaradas.
As variaveis const nao podem ser redefinidas.
Variaveis const tem escopo de bloco.

As variaveis const devem receber o valor quando forem declaradas

const PI = 3.14159265359; esta e a forma correta

const PI;
PI = 3.14159265359; forma incorreta

Quando usar JavaScript const? Sempre declare uma variável com const quando souber que o valor não deve ser alterado.
